# Agents Directory

Custom agents for this workspace live in this visible `agents/` directory.

## Why this location
For repository readability, agent definitions are maintained in `agents/*.agent.md`.

This is why `Negativenagesh` is defined at:
- `agents/negativenagesh.agent.md`

## Skills vs Agents
- Agents define persona, tools, and orchestration behavior.
- Skills define on-demand workflows and are maintained in `skills/` for this repository.
