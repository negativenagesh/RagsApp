# Skills Directory

This project uses Agent Skills under this visible `skills/` directory.

## Available Skills

- `rag`: Document-grounded retrieval and citation workflow.
- `arecanut-price`: Live arecanut mandi price fetch with source-backed response.
- `news-search`: DuckDuckGo-backed latest news retrieval with language-preserving responses.

## Notes

- Keep each skill self-contained.
- Put reusable references under each skill's `references/` folder.
- Ensure `name` in `SKILL.md` matches the skill folder name.
